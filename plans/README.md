- **`useApprovePendingAlbumItems` unawaited batch updates** (`src/hooks/albumItems/pendingItems/useApprovePendingAlbumItems.ts:43`): `WriteBatch.update` queues synchronously, so the unawaited promises are benign; cosmetic only.
